# 执行 BASE PRP

使用PRP文件实现功能。

## PRP文件: $ARGUMENTS

## 执行流程

1. **加载PRP**
   - 读取指定的PRP文件
   - 理解所有上下文和需求
   - 遵循PRP中的所有指令并根据需要扩展研究
   - 确保拥有实现PRP所需的全部上下文
   - 根据需要进行更多网络搜索和代码库探索

2. **深度思考**
   - 执行计划前认真思考。制定全面计划解决所有需求。
   - 将复杂任务分解为更小、可管理的步骤，使用待办事项工具。
   - 使用TodoWrite工具创建和跟踪实施计划。
   - 识别现有代码中的实现模式以遵循。

3. **执行计划**
   - 执行PRP
   - 实现所有代码

4. **验证**
   - 运行每个验证命令
   - 修复任何失败
   - 重新运行直到全部通过

5. **完成**
   - 确保所有检查项已完成
   - 运行最终验证套件
   - 报告完成状态
   - 再次阅读PRP确保已实现所有内容

6. **引用PRP**
   - 如有需要，随时可以再次引用PRP

注意：如果验证失败，使用PRP中的错误模式修复并重试。
